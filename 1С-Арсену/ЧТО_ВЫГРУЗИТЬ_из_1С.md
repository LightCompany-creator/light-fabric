# Что выгрузить из 1С для LightFabric

> Куда класть готовые файлы: папка `1С-Арсену/выгрузки-из-1С/` (рядом с этим файлом).
> Формат: **Excel (.xlsx)**, первая строка — заголовки колонок.
> Главное правило: в КАЖДОЙ выгрузке должен быть **внутренний код 1С** элемента —
> это ключ, по которому LightFabric состыкуется с 1С. Без кода данные не подключить.

---

## 1. Номенклатура → `1c-номенклатура.xlsx`
Товары производства + материалы/сырьё (всё, что участвует в выработке и списании).

| Колонка | Пример |
|---|---|
| код 1С | 00-00001234 |
| артикул | 112/н |
| наименование | Сапоги женские ЭВА с манжетой |
| тип | Товар / Материал |
| единица измерения | пар / кг / м / шт |
| группа | Готовая продукция/Обувь |

## 2. Сотрудники → `1c-сотрудники.xlsx`
Физлица, работающие в производстве.

| Колонка | Пример |
|---|---|
| код 1С | 00-0000056 |
| табельный номер | 0056 |
| ФИО | Иванова Анна Петровна |
| код подразделения | 00-000003 |
| должность | Литейщик |

## 3. Подразделения → `1c-подразделения.xlsx`
Цеха / подразделения (сопоставим с нашими цехами один раз вручную).

| Колонка | Пример |
|---|---|
| код 1С | 00-000003 |
| наименование | Литейный цех |
| код родителя | (пусто, если верхний уровень) |

## 4. Склады → `1c-склады.xlsx`

| Колонка | Пример |
|---|---|
| код 1С | 00-000001 |
| наименование | Склад сырья |

## 5. Организации → `1c-организации.xlsx`
Юр. лица (для шапки документов).

| Колонка | Пример |
|---|---|
| код 1С | 00-000001 |
| наименование | ИП Хачатуров А.В. |
| ИНН | 262800000688 |

## 6. Виды расчёта → `1c-виды-расчёта.xlsx`
Виды начислений ЗП (нужно, чтобы правильно проводить сдельную оплату).

| Колонка | Пример |
|---|---|
| код 1С | 00-000007 |
| наименование | Сдельная оплата |
